Comment fonctionne l'import ?

  • L'application détecte automatiquement les colonnes — pas besoin que le CSV ait un format précis
  • Elle cherche dans chaque ligne une référence structurée au format +++XXX/XXXX/XXXXX+++
  • Le matricule FASE est extrait de la référence et comparé aux élèves de la base de données
  • Si l'élève est trouvé → son portefeuille est crédité du montant du virement
  • Les lignes sans référence structurée reconnue sont ignorées silencieusement
Vérifiez avant d'importer Chaque import crédite les portefeuilles immédiatement. Un double import du même fichier créditerait les élèves deux fois. Vérifiez que vous n'avez pas déjà importé cet extrait.
